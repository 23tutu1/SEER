Abstract

神经渲染在 3D 重建中取得了显著成就，但其缺乏对场景语义的深入理解。SEER 提出了一种基于大模型语义先验的新架构， 通过 Transformer 结构有效融合空间几何与语义信息。我们的方法在多项基准测试中达到了最先进的性能，并支持高度自由的场景编辑。

Methodology

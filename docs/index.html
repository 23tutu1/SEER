SEER 利用大模型 (LLM/VLM) 的语义先验，实现了对 3D 场景的高精度语义控制渲染。

摘要 (Abstract)

神经渲染在 3D 重建中取得了显著成就，但其缺乏对场景语义的深入理解，限制了编辑的可控性。在本文中，我们提出了 SEER (Semantic Embedded Expressive Rendering)，一个利用预训练大模型语义先验的新型框架。我们证明了通过将大模型的密集语义特征嵌入神经辐射场，可以在保持几何一致性的同时，实现高度灵活的属性编辑和视角合成。

SemPrior-Rend: Semantic Prior Driven Controllable Neural Rendering

Code Report Demo

Semantic Prior

用 VLM/CLIP 语义对齐，引导渲染结果符合文本描述。

Multi-view Consistency

跨视角语义一致性损失，减少视角切换闪烁与漂移。

Local Editing (Optional)

掩码 + 局部 prompt，实现局部语义编辑与控制。

Abstract

我们在 NeRF 优化中引入视觉语言模型（如 CLIP）语义先验，通过全局语义对齐与跨视角一致性约束， 实现文本可控的神经渲染，并可扩展到局部掩码编辑。该项目以课程作业为目标，强调可复现与工程完整性。

Method

训练目标由像素重建与语义约束共同组成：
L = Lrgb + λgLsemglobal + λmvLmv (+ λlLsemlocal)

Global Semantic

对整张渲染图与文本 prompt 做 cosine 对齐，促进整体语义一致。

Multi-view

同一 batch 内不同视角的语义 embedding 约束一致，降低闪烁与漂移。

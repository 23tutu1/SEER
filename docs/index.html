Abstract

本研究引入视觉语言模型（如 CLIP）作为语义先验，在 NeRF 优化中加入全局语义对齐与多视角一致性约束， 实现文本可控的神经渲染；并可扩展到掩码 + 局部提示的局部编辑控制。

Method

训练目标由像素重建与语义约束共同组成： L = Lrgb + λgLsemglobal + λmvLmv (+ λlLsemlocal)

Global Semantic

渲染图与文本 prompt 做 cosine 对齐，增强整体语义一致性。

Multi-view Consistency

同 batch 不同视角 embedding 一致，减少闪烁与漂移。

Local Editing

掩码区域与局部 prompt 对齐，实现局部可控编辑。

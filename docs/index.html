Introduction

现有 NeRF 在语义可控编辑中常出现跨视角不一致与背景扰动。SEER 将大模型语义先验蒸馏到可渲染的语义场中， 并通过条件调制将文本/语义控制仅作用于外观分支，同时用软掩码约束局部编辑区域，从而实现稳定的对象级可控神经渲染。

SEER框架总体流程

SEER的总体流程展示了如何将多视角图像与位置信息输入CLIP视觉编码器，并通过射线网络与语义特征结合，生成可渲染的语义场。此过程中的语义条件调制确保了外观的可控渲染。

Method

SEER的核心由三部分组成： 可渲染语义场蒸馏 语义条件调制 软掩码局部控制。

下方展示了模型的流程与每个步骤的详细工作原理。

语义蒸馏闭环

语义蒸馏闭环的过程首先通过CLIP提取的语义特征与射线网络结合，实现一致的可渲染语义场。在这个过程中，语义信息对齐是确保高质量渲染的关键。

语义条件调制

语义条件调制部分通过将条件信息注入到外观分支，实现对渲染外观的控制。此部分采用了FiLM模块，以确保外观特征的可控性，同时保持场景的几何结构不变。

两阶段训练策略

为了提高训练稳定性，SEER采用了两阶段训练策略。第一阶段专注于几何结构和外观信息的学习，第二阶段则引入语义调制和局部控制，以确保渲染结果的准确性和可控性。
